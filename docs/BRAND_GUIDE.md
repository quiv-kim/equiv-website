# EQUIV M&A Brand Guide

## Brand Positioning

EQUIV M&A는 기업의 적정 가치를 발견하고, 균형 잡힌 거래를 설계하는 프리미엄 M&A Advisory Firm이다.

핵심 관점:

- 기업을 단순히 거래하지 않는다.
- 기업의 가치를 분석하고 설계한다.
- 모든 기업에는 고유한 가치가 있다.
- 좋은 M&A는 올바른 순서와 맞춤형 전략에서 시작된다.

## Brand Philosophy

Korean:

모든 기업에는 가치가 있습니다.

EQUIV는 그 가치를 성공적인 거래로 연결합니다.

Every Company Has Value.

EQUIV Connects Value to Opportunity.

## Pricing Communication Principle

EQUIV는 가격으로 선택받는 브랜드가 아니라,
전문성과 거래 전략으로 선택받는 M&A Advisory 브랜드를 지향한다.

홈페이지에서는 다음 내용을 전면적으로 노출하지 않는다.

- 착수금
- 성공보수
- 수수료율
- 할인
- 최저 수수료
- 무료 상담 강조
- 파트너 수수료
- 리베이트 또는 금전적 혜택

가격은 브랜드의 경쟁력이 아니라,
기업의 규모,
거래 구조,
업무 범위,
프로젝트의 특성에 따라
함께 설계되는 자문 범위의 결과이다.

홈페이지는 가격을 설명하는 공간이 아니라,
브랜드 철학,
전문성,
업무 방식,
신뢰를 전달하는 공간으로 사용한다.

상담 이후
기업의 상황과 목적을 충분히 이해한 후
가장 적합한 자문 범위와 진행 방식을 제안한다.

## The EQUIV Mark

공식 심볼명은 `The EQUIV Mark`이며 공식 Primary Logo는 `[The EQUIV Mark] + Vertical Divider + EQUIV` Lockup이다.

The EQUIV Mark는 M&A의 `M`을 출발점으로, 균형 있는 가치(Equivalence)를 만나 EQUIV의 `E`로 완성되는 순간을 상징한다. 모든 거래는 M&A에서 시작되고 성공적인 거래는 균형 있는 가치에서 완성된다는 브랜드 철학을 담는다.

사용 원칙:

- 기본 로고에는 `M&A ADVISORY`, `M&A Brokerage` 등의 Descriptor를 사용하지 않는다.
- Mark는 Premium Gold, Wordmark와 Divider는 배경에 따라 Navy 또는 White를 사용한다.
- 웹 Header에서는 Mark를 실제 SVG 이미지로 표시하고 Wordmark는 가시성을 보장하는 `EQUIV` Text Wordmark로 렌더링한다.
- 비율, 정렬과 요소 순서를 바꾸지 않는다.
- Header, Footer와 Consultation Modal에서 동일한 Lockup을 사용한다.
- Favicon은 The EQUIV Mark 단독 심볼을 사용한다.
- Flat Gold를 기본으로 하며 과도한 Gradient, Shadow와 장식을 적용하지 않는다.

## Color System

### Navy

- Primary brand color
- 신뢰, 전문성, 금융 자문사의 안정감을 표현
- CSS variable: `--color-primary`
- Current value: `#071A2F`

### Gold

- Accent color
- 가치, 균형, 프리미엄 감도를 표현
- CSS variable: `--color-gold`
- Current value: `#A98235`

### White

- Main background
- 여백과 정제된 인상을 만드는 기본 색상
- CSS variable: `--color-white`
- Current value: `#FFFFFF`

### Light Gray

- Muted section background
- 상세페이지 Hero 또는 Process 배경에 사용
- CSS variable: `--color-gray`
- Current value: `#F5F6F8`

### Text / Muted

- Main text: `#222222`
- Muted text: `#6F7782`

## Tone & Manner

EQUIV의 문체는 압박하거나 과장하지 않는다. 전문성과 신뢰를 바탕으로, 기업 대표가 자연스럽게 상담을 검토하도록 돕는다.

지향하는 톤:

- Premium
- Calm
- Strategic
- Trustworthy
- Minimal
- Advisory-oriented

피해야 할 톤:

- 지나치게 영업적인 표현
- 과도한 성공 보장 표현
- 자극적인 투자/매각 유도 문구
- 불필요하게 장황한 설명
